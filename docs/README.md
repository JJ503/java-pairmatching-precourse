## 미션 흐름에 따른 문제 이해

1. 크루 이름 파일 입력 받기
    1. Frontend crew 목록 받기
    2. Backend crew 목록 받기


2. 프로그램 기능 선택
    1. 기능 목록 출력
       ```
       1. 페어 매칭
       2. 페어 조회
       2. 페어 초기화
       Q. 종료
       ```
    2. 선택 기능 입력
    3. 입력 값 유효성 검증
        - 값이 있는가?
        - 1, 2, 3, Q 값 중 하나인가?
        - 위 조건에 부합하지 않는 경우

          → `IllegalArgumentException`
          를 발생시키고, `[ERROR]`
          로 시작하는 에러 메시지를 출력 후 해당 부분부터 다시 입력 받기


3. 페어 매칭 기능
    1. 과정과 미션 출력
    2. 매칭하고자 하는 과정, 레벨, 미션 입력 받기
    3. 크루 페어 매칭하기
        - 페어 매칭 조건
            - 미션을 함께 수행할 페어를 두명씩 매칭한다
            - 페어 매칭 대상이 홀수인 경우 한 페어는 3인으로 구성한다. (마지막 남은 크루는 마지막 페어에 포함)
            - 같은 레벨에서 이미 페어를 맺은 크루와는 다시 페어로 매칭될 수 없다.
            - 3회 시도까지 매칭이 되지 않거나 매칭을 할 수 있는 경우의 수가 없으면 에러 메시지를 출력한다.
        - 페어 재미칭 시도
            - 이미 페어 매칭은 한 경우
            - 안내 문구를 출력 후 매칭을 진행한다.
            - 아니오를 선택할 경우 코스, 레벨, 미션을 다시 선택한다.
                - 예시

                    ```
                    매칭 정보가 있습니다. 다시 매칭하시겠습니까?
                    네 | 아니오
                    아니오
                    ```

        - 출력되는 페어의 순서는 `camp.nextstep.edu.missionutils.Randoms`
          의 shuffle 메서드의 결과 순서로 정렬
            - 예시

                ```
                페어 매칭 결과입니다.
                용팔 : 대만
                대협 : 덕규
                치수 : 준호
                태웅 : 백호
                달재 : 태산
                한나 : 수겸
                태섭 : 대남
                준섭 : 소연
                현준 : 호열
                구식 : 경태
                ```
<br>

4. 페어 조회 기능
    1. 과정과 미션 출력
    2. 조회하고자 하는 과정, 레벨, 미션 입력 받기
    3. 조회하고자 하는 페어 매칭 결과 출력하기
    - 페어 정보가 없는 경우 예시

        ```
        [ERROR] 매칭 이력이 없습니다.
        ```

    - 페어 정보가 있는 경우 예시

        ```
        #############################################
        과정: 백엔드 | 프론트엔드
        미션:
          - 레벨1: 자동차경주 | 로또 | 숫자야구게임
          - 레벨2: 장바구니 | 결제 | 지하철노선도
          - 레벨3: 
          - 레벨4: 성능개선 | 배포
          - 레벨5: 
        ############################################
        과정, 레벨, 미션을 선택하세요.
        ex) 백엔드, 레벨1, 자동차경주
        프론트엔드, 레벨1, 자동차경주
        
        페어 매칭 결과입니다.
        이브 : 윌터
        보노 : 제키
        신디 : 로드
        제시 : 린다
        시저 : 라라
        니콜 : 다비
        리사 : 덴버 : 제키
        ```

<br>

5. 페어 초기화 기능
    1. 모든 페어 매칭 결과 기록 초기화


6. 종료 기능
    1. 프로그램 종료

<br>

## 기능에 따른 분류
### PairMatchingController

> 페어 매칭 프로그램의 절차를 관리하는 클래스

- [ ]  객체 초기화
- [ ]  크루 이름 파일 입력 받기
    - [ ]  파일 선택 후 EnterCrewFile에 전달
        - [ ]  백엔드, 프론트엔드 각각 입력
        - [ ]  크루 객체에 각각 저장
- [ ]  기능 선택 입력 받기
    - [ ] 입력 값 검증
      - [ ]  값이 있는지 확인
      - [ ]  1, 2, 3, Q 값 중 하나인지 확인
      - [ ]  위 사항에 부합하지 않는 경우 예외 발생
          - 값이 없는 경우 : `[ERROR] 값을 입력해 주시길 바랍니다.`
          - 기능 외 값을 입력한 경우 : `[ERROR] 기능 목록 중 하나를 선택해 주시길 바랍니다 (1, 2, 3, Q 중 선택).`
- [ ]  1을 입력한 경우 → 페어 매칭
- [ ]  2를 입력한 경우 → 페어 조회
- [ ]  3을 입력한 경우 → 페어 초기화
- [ ]  Q를 입력한 경우 → 프로그램 종료

### EnterCrewFile

> 파일로 저장되어 있는 크루 이름 목록을 불러오는 클래스

- [x]  파일 읽기
- [x]  파일 내용을 List<String> 형으로 반환
    - 각 라인은 crew 이름으로 List에 추가해 반환할 수 있도록 한다.

### Menu

> 메뉴 목록에 대한 정보 제공과 입력 값을 검증해주는 enum

- [x]  기능의 종류를 출력 형식으로 문자열 반환


### Course

> 과정에 대한 정보 제공과 입력 값을 검증해주는 enum

- [ ]  입력 값 검증
    - [ ]  값이 있는지 확인
    - [ ]  존재하는 과정인지 확인
    - [ ]  위 조건에 부합하지 않는 경우
        - 값이 없는 경우

          `[ERROR] 값을 입력해 주시길 바랍니다.`

        - 과정이 존재하지 않는 경우

          `[ERROR] 해당 과정을 찾지 못했습니다.`


### Level

> 레벨과 미션에 대한 정보 제공과 입력 값을 검증해주는 enum

- [ ]  입력 값 검증
    - [ ]  값이 있는지 확인
    - [ ]  존재하는 레벨인지 확인
    - [ ]  존재하는 미션인지 확인
    - [ ]  위 조건에 부합하지 않는 경우
        - 값이 없는 경우

          `[ERROR] 값을 입력해 주시길 바랍니다.`

        - 레벨이 존재하지 않는 경우

          `[ERROR] 해당 레벨을 찾지 못했습니다.`

        - 미션이 존재하지 않는 경우

          `[ERROR] 해당 미션을 찾지 못했습니다.`


### PairMatching

> 페어 매칭 기능을 수행하고 관리하는 클래스

- [ ]  크루 이름 목록 랜덤으로 섞기

  → ``camp.nextstep.edu.missionutils.Randoms``의 shuffle 메서드를 활용

- [ ]  앞에서부터 순서대로 두명씩 페어 맺기
    - [ ]  홀수인 경우 마지막 사람을 마지막 페어에 포함시키기
    - [ ]  같은 레벨에서 이미 페어로 만난적이 있는 크루가 다시 페어로 매칭된 경우

      → 크루 목록의 순서를 다시 랜덤으로 섞어서 매칭을 시도

    - [ ]  3회 시도까지 매칭되지 않는 경우 or 매칭을 할 수 있는 경우의 수가 없는 경우

      → 에러 메시지를 출력

      `[ERROR] 페어 매칭을 할 수 없습니다.`

- [ ]  페어 매칭을 한 기록이 있는 경우
    - [ ]  페어 재매칭 안내 문구 출력
    - [ ]  페어 재매칭 여부 선택 입력
    - [ ]  네를 선택한 경우
        - [ ]  페어 매칭 기능 수행
    - [ ]  아니오를 선택한 경우
        - [ ]  코스, 레벨, 미션 다시 선택
